# di-brand-kit

Центральный бренд-кит веб-домашек «Математика с Ди и Леммой».
Источник стилей для всех домашек/интерактивов. Подключение:

```html
<link rel="stylesheet" href="https://dmitrydruzhkovv-arch.github.io/di-brand-kit/brand-kit.css">
```

Мастер-копия живёт в монорепо `💻 Веб-проекты/_бренд-кит.css`; этот репо — её прод-деплой. Правит Лемма, без ПДн.
